# footsteps
Comp 324 - Client side development group project 

--- 

Install necessary dependencies on fresh repository clones, run: 

### `npm install`

--- 

At the root project directory, run:

### `npm run serve` or `nodemon serve.js`

<br>

to start the server and get the application up and running. <br>
the app will be running on **localhost:3000**

---

## Technologies used

1. Node.js 
2. Body Parser 
3. Express 
4. MongoDB 
5. Mongoose
